# CSS Magic & WCAG 3 Dashboard

An interactive dashboard showcasing modern CSS features while demonstrating WCAG 3 accessibility compliance.

## Features

- **CSS Grid Layout**: Demonstrates grid areas and responsive design
- **CSS Flexbox**: Shows flexible layouts that adapt to content
- **CSS Animations**: Smooth transitions with motion preference respect
- **CSS Custom Properties**: Dynamic theming and color customization
- **WCAG 3 Compliance**: Semantic HTML, ARIA labels, keyboard navigation, and accessibility features

## WCAG 3 Guidelines Addressed

- **Perceivable**: High contrast, reduced motion support
- **Operable**: Keyboard navigation, focus management
- **Understandable**: Clear labeling and logical content flow
- **Robust**: Semantic HTML and ARIA support

## Technologies Used

- HTML5 with semantic elements
- CSS3 with modern features (Grid, Flexbox, Custom Properties, Animations)
- Vanilla JavaScript for interactivity
- Responsive design with media queries

## Setup

1. Clone this repository
2. Open `index.html` in a modern web browser
3. For GitHub Pages deployment, enable Pages in repository settings

## Browser Support

- Modern browsers with CSS Grid and Flexbox support
- Graceful degradation for older browsers

## Accessibility Features

- ARIA labels and roles
- Keyboard navigation support
- High contrast themes
- Reduced motion preferences
- Semantic HTML structure
